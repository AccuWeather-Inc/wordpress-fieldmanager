Fieldmanager is a comprehensive toolkit for developers to build fields, forms, meta boxes, custom admin screens, and more.

== Description ==

Fieldmanager is a powerful library which can make the development of sophisticated features a breeze. To get started, simply install this plugin into your plugins directory and activate it on the plugins screen. To learn how to use Fieldmanager's API, visit the project's official website at [Fieldmanager.org](http://fieldmanager.org). There is also [a demo plugin](https://github.com/alleyinteractive/fieldmanager-demos) which illustrates a lot of what you can do with Fieldmanager.

= Documentation =

See http://fieldmanager.org/ for full documentation.

= Bug Reports & Feature Requests =

= Contribute =

All development happens on [GitHub](https://github.com/alleyinteractive/wordpress-fieldmanager). Pull requests welcome!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/fieldmanager` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Start coding. See http://fieldmanager.org/ for more.

== Frequently Asked Questions ==

= Is there a UI for building fields? =

There is not. Fieldmanager is an API for developers to use to build fields.

= How can I do _______ with Fieldmanager? =

First, [check the docs](http://fieldmanager.org/). If you can't find your specific need, Fieldmanager is an extremely flexible framework and is very easy to extend to do what you need.
